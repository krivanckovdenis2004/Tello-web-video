# Tello Video

Простой сайт для видеозвонков по коду комнаты.

## Как работает

1. Нажимаешь **Создать комнату**.
2. Получаешь код и ссылку.
3. Отправляешь другу.
4. Друг открывает ссылку или вводит код.
5. Начинается видеозвонок через Jitsi Meet.

## Файлы

- `index.html` — главная страница
- `style.css` — дизайн
- `script.js` — логика создания комнат и подключения Jitsi
