# Tello Video v2

Сайт для быстрых видеозвонков по ссылке.

## Что изменено

- Кнопка «Скопировать ссылку».
- Кнопка «Поделиться».
- Автоматический вход по ссылке `?room=TELLO-000000`.
- Меньше текста на главном экране.
- Экран ожидания друга.
- Кнопки микрофона, камеры, выхода и смены камеры.
- Резервный видеосервер: если не подключается, нажмите «Если не подключается».

## Как пользоваться

1. Нажмите «Создать звонок».
2. Отправьте ссылку другу.
3. Нажмите «Войти в звонок».
4. Друг открывает ту же ссылку и тоже нажимает «Войти».

Если один видит только себя — проверьте, что оба открыли одну и ту же ссылку и один и тот же код комнаты.
